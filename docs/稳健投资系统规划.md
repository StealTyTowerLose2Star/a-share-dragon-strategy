# 中长期稳健投资选股系统 - 需求规划

## 📋 项目概述

**仓库名称**：`a-share-steady-investment`（待创建）

**定位**：管理总仓位的 80%，进行中长期稳健投资

**风险等级**：中低风险

**预期收益**：年化 15-30%

**持有周期**：1-6 个月

---

## 🎯 核心策略模块

### 1. 价值投资选股模块

**核心逻辑**：
- 低估值：PE/PB 低于行业平均 30% 以上
- 高 ROE：连续 3 年 ROE > 15%
- 稳定分红：股息率 > 3%，连续 5 年分红
- 业绩增长：净利润增速 > 10%

**数据需求**：
- 财务报表（年报/季报）
- 估值指标（PE/PB/PS）
- 分红历史
- 行业数据

**筛选流程**：
```python
1. 全市场筛选 ROE > 15%
2. 过滤 PE < 行业平均
3. 检查分红记录
4. 验证业绩增长
5. 生成候选股票池
```

**仓位建议**：稳健仓位的 40-50%（总仓位 32-40%）

---

### 2. 趋势跟踪系统

**核心逻辑**：
- 均线多头排列：5/10/20/60 日线依次向上
- 中期趋势：20 日线斜率 > 0
- 成交量：温和放大（相对 5 日均量 +20%）
- 突破信号：突破关键阻力位

**技术指标**：
- MA5/10/20/60
- MACD（金叉/死叉）
- RSI（超买/超卖）
- 成交量

**买入信号**：
```python
if 均线多头 and MACD 金叉 and 成交量放大:
    买入信号 = True
```

**卖出信号**：
```python
if 跌破 20 日线 or MACD 死叉:
    卖出信号 = True
```

**仓位建议**：稳健仓位的 20-30%（总仓位 16-24%）

---

### 3. 行业轮动策略

**核心逻辑**：
- 宏观经济周期判断（美林时钟）
- 行业景气度分析
- 资金流向监测
- 政策导向跟踪

**周期阶段**：
| 经济阶段 | 推荐行业 |
|----------|----------|
| 复苏期 | 金融、科技、可选消费 |
| 过热期 | 资源、能源、工业 |
| 滞胀期 | 必需消费、医疗、公用事业 |
| 衰退期 | 债券、高股息、防御性 |

**数据需求**：
- GDP 增速
- CPI/PPI
- PMI 数据
- 行业景气指数
- 北向资金流向

**仓位建议**：稳健仓位的 20-30%（总仓位 16-24%）

---

### 4. 高股息策略

**核心逻辑**：
- 股息率 > 5%
- 连续 5 年稳定分红
- 现金流充沛（经营现金流/净利润 > 1）
- 低负债率（资产负债率 < 60%）

**筛选标准**：
```python
if 股息率 > 5% and 
   连续分红年数 >= 5 and 
   现金流比率 > 1 and 
   负债率 < 60%:
    加入股票池
```

**适用场景**：
- 市场震荡期（防御）
- 退潮期（替代债券）
- 长期持有（复利）

**仓位建议**：稳健仓位的 10-20%（总仓位 8-16%）

## 📈 风控规则

### 单只股票风控

**上限**：
- 单只股票 ≤ 总仓位 15%
- 单一行业 ≤ 总仓位 30%

**止损**：
- 价值投资：基本面恶化时止损
- 趋势跟踪：跌破 20 日线止损（-8%）
- 行业轮动：行业逻辑破坏时止损
- 高股息：股息率降至 3% 以下时调整

### 总体风控

**最大回撤控制**：
- 单月回撤 > 10%：降至 50% 仓位
- 单月回撤 > 15%：降至 30% 仓位
- 单月回撤 > 20%：清仓观望

**仓位上限**：
- 正常情况：80-95%
- 高风险时期：50-70%
- 极端行情：30-50%

- 独立仓库
- 独立数据
- 独立风控
- 独立运行

### 仓位协调

**总仓位控制**：
```
总可用仓位 = 龙头战法仓位 + 稳健投资仓位
           = 20% + 80%
           = 100%
```

**极端行情协调**：
- 龙头战法退潮期：仓位降至 0-4%
- 稳健投资可适度增加：80% → 90%
- 总仓位仍保持谨慎：90-94%
