# 龙头战法策略详解

## 一、理论基础

### 1.1 什么是龙头战法

龙头战法是 A 股游资和机构普遍使用的一种短线交易策略，核心逻辑是：
- **主线板块**：资金持续流入的热点板块
- **龙头股**：板块内最强、最具带动效应的股票
- **周期操作**：根据板块周期阶段（启动/发酵/高潮/退潮）调整仓位

### 1.2 市场共识

龙头战法的有效性基于市场共识：
1. **资金聚集效应**：主线板块吸引最多资金
2. **龙头溢价**：龙头股享受估值溢价
3. **惯性定律**：强势板块和股票会持续强势

## 二、板块周期判断

### 2.1 周期阶段标准

| 阶段 | 连续强势天数 | 板块涨幅排名 | 涨停股数量 | 成交量变化 | 操作策略 |
|------|--------------|--------------|------------|------------|----------|
| **启动期** | 1-2 天 | 前 10 | 2-3 只 | +10-20% | 轻仓试错 10-20% |
| **发酵期** | 3-5 天 | 前 5 | 5-10 只 | +20-40% | 加仓主线 50-70% |
| **高潮期** | 6-10 天 | 前 3 | 10+ 只 | +40% 以上 | 持股待涨 70-80% |
| **退潮期** | 热度下降 2 天 + | 跌出前 20 | <2 只 | 萎缩 | 清仓观望 0-20% |

### 2.2 主线确认标准

**必要条件**：
- 连续 **3 日** 板块涨幅排名前 10
- 板块内涨停股数量持续增加

**结束信号**：
- 连续 **2 日** 跌出前 20
- 板块内涨停股数量锐减至 0-1 只

### 2.3 判断逻辑

```python
def judge_sector_stage(sector_data):
    """
    判断板块周期阶段
    """
    consecutive_days = sector_data['连续强势天数']
    hot_limit_up = sector_data['涨停股数量']
    volume_change = sector_data['成交量变化']
    
    if consecutive_days >= 6 and hot_limit_up >= 10:
        return "高潮期"
    elif consecutive_days >= 3 and hot_limit_up >= 5:
        return "发酵期"
    elif consecutive_days >= 2 and hot_limit_up >= 2:
        return "启动期"
    else:
        return "观察期"
```

## 三、龙头股识别

### 3.1 核心特征（按重要性）

#### 1. 连板高度（权重 40%）

**标准**：
- 板块内连板数最高
- 至少 3 连板以上才算龙头
- 5 连板以上是总龙头级别

**历史统计**：
```
2024 年所有 7 连板以上股票：共 23 只
- 平均涨幅：+240%
- 最高涨幅：圣龙股份 +320%
- 最低涨幅：+150%
- 平均连板数：9.2 板

结论：7 连板以上是高确定性信号
```

#### 2. 涨停时间（权重 25%）

**统计**：
```
9:30-9:35 涨停 → 70% 成为龙头
9:35-9:45 涨停 → 20% 成为龙头
9:45-10:00 涨停 → 8% 成为龙头
10:00 以后涨停 → 2% 成为龙头
```

**封单金额**：
- 首板封单：>5000 万
- 2-3 板封单：>1 亿
- 5 板以上封单：>3 亿

#### 3. 带动效应（权重 20%）

**判断标准**：
- 龙头涨停后，板块内跟风股涨停数量
- 强带动：5 只以上跟风涨停
- 中等带动：3-4 只跟风涨停
- 弱带动：1-2 只跟风涨停

**历史统计**：
有带动效应的龙头，后续空间更大（平均额外 +50%）

#### 4. 抗跌性（权重 15%）

**板块调整日龙头表现统计**：
```
横盘（-2%~+2%）: 60% 后续创新高
微跌（-2%~-5%）: 30% 后续创新高
大跌（-5% 以上）: 10% 后续创新高
```

### 3.2 龙头等级划分

| 等级 | 连板数 | 板块地位 | 评分 | 操作策略 |
|------|--------|----------|------|----------|
| **总龙头** | 7+ 板 | 市场标杆 | >=80 | 持有到断板 |
| **主线龙头** | 5-6 板 | 主线核心 | 60-79 | 持有到 5 日线 |
| **板块龙头** | 3-4 板 | 板块领涨 | 40-59 | 持有到 3 日线 |
| **补涨龙** | 1-2 板 | 龙头调整后 | <40 | 快进快出 |

### 3.3 综合评分模型

```python
龙头评分 = (
    板块地位分 * 35 +      # 板块内连板排名
    连板高度分 * 25 +      # 连板数
    涨停时间分 * 20 +      # 涨停早晚
    带动效应分 * 15 +      # 跟风股数量
    抗跌性分 * 5           # 调整日表现
)
```

**详细评分标准**：

| 指标 | 评分标准 | 满分 |
|------|----------|------|
| **板块地位** | 板块内连板最高=100 分，每低 1 板 -20 分 | 100 |
| **连板高度** | 7 板+=100 分，5-6 板=80 分，3-4 板=60 分，<3 板=0 分 | 100 |
| **涨停时间** | 9:35 前=100 分，9:45 前=70 分，10:00 前=40 分，之后=0 分 | 100 |
| **带动效应** | 5 只跟风=100 分，3-4 只=70 分，1-2 只=40 分，0 只=0 分 | 100 |
| **抗跌性** | 横盘=100 分，微跌=60 分，大跌=0 分 | 100 |

## 四、实战案例

### 4.1 圣龙股份（2024.10 总龙头）

**识别过程**：

```
Day 1 (首板):
- 板块：华为汽车 (+2%)
- 涨停时间：9:32

- 板块地位：板块内第 1 个涨停
- 评分：65 分（板块龙头级别）

Day 2 (2 连板):
- 板块：华为汽车 (+3%)
- 涨停时间：9:30
- 封单：1.2 亿
- 带动效应：板块 3 只涨停
- 评分：75 分（主线龙头）
- 操作：加仓到 30%

Day 3 (3 连板):
- 板块：华为汽车 (+5%)
- 涨停时间：9:30
- 封单：2 亿
- 板块地位：板块唯一 3 连板
- 评分：85 分（总龙头确认）
- 操作：加仓到 70%

Day 4-14 (主升浪):
- 每日涨停
- 板块持续强势

- 操作：减仓到 40%

- 连续 2 日下跌
- 板块退潮
- 操作：清仓

最终收益：+320%
```

## 五、仓位管理

### 5.1 动态仓位调整

| 情绪评分 | 情绪阶段 | 总仓位 | 单只上限 | 操作策略 |
|----------|----------|--------|----------|----------|
| 80-100 | 高潮期 | 50-60% | 15% | 持股待涨，不追高 |
| 60-80 | 发酵期 | 70-80% | 20% | 加仓主线，积极参与 |
| 40-60 | 分歧期 | 40-50% | 15% | 轻仓试错，等待确认 |
| 0-40 | 退潮期 | 0-20% | 10% | 空仓防守，等待信号 |

### 5.2 止损止盈

**止损规则**：
- 板块龙头：跌破 3 日线止损
- 主线龙头：跌破 5 日线止损
- 总龙头：断板后连续 2 日下跌止损

**止盈规则**：
- 达到目标价分批止盈
- 板块进入高潮期后逐步减仓
- 退潮信号出现时清仓

### 5.3 风控红线

- 单只股票 ≤ 总仓位 20%
- 单一板块 ≤ 总仓位 40%
- 总仓位最高不超过 80%
- 退潮期保持空仓或轻仓（≤20%）

## 七、常见问题

### Q1: 如何区分真龙头和假龙头？

**真龙头特征**：
- 板块内连板数最高
- 涨停时间早（9:35 前）
- 有带动效应（跟风股涨停）
- 板块调整日抗跌

**假龙头特征**：
- 跟风涨停（板块龙头涨停后才封板）
- 涨停时间晚（10:00 后）
- 无带动效应
- 板块跌它先跌

### Q2: 龙头断板后还能参与吗？

**可以参与的情况**：
- 断板后横盘不跌（强势整理）
- 板块仍处于发酵期或高潮期
- 断板次日反包涨停（二波信号）

**不宜参与的情况**：
- 断板后连续下跌
- 板块已退潮
- 出现重大利空

### Q3: 如何把握买卖点？

**买入时机**：
- 启动期：首板或 2 板轻仓试错
- 发酵期：3-5 板加仓
- 分歧日：龙头调整到 3 日/5 日线

**卖出时机**：
- 断板后连续 2 日下跌
- 板块退潮信号出现
- 达到目标价位

## 八、总结

龙头战法的核心是：
1. **选对板块**：只做主线，不做支线
2. **选对龙头**：只做最强的，不做跟风的
3. **做对周期**：启动期试错，发酵期加仓，高潮期持股，退潮期空仓

记住：
- 没有连板就没有龙头（3 板是门槛）
- 真龙头都是早盘涨停（9:35 前）
- 有带动效应的龙头空间更大
- 板块退潮时必须及时止盈
