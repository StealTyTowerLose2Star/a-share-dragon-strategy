## 🎯 核心策略模块

### 1. 趋势轮动模块（40-50% 仓位）

**核心逻辑**：
- 识别市场主线板块
- 资金流向监测
- 板块轮动节奏把握
- 主线切换时及时调仓

**关键指标**：
- 板块涨幅排名（连续 N 日前 10）
- 板块成交量变化（相对 5 日均量）
- 板块内涨停股数量
- 北向资金行业流向

**筛选流程**：
```python
1. 获取全市场板块涨幅排名
2. 筛选连续 3 日前 10 的板块
3. 检查板块成交量是否放大
4. 统计板块内涨停股数量
5. 生成轮动股票池（每个板块 3-5 只）
6. 选择板块内趋势最强的股票
```

**买入信号**：
```python
if 板块连续 3 日前 10 and 
   成交量放大 > 20% and 
   涨停股数量 >= 3 and 
   个股趋势向上:
    买入信号 = True
```

**卖出信号**：
```python
if 板块连续 2 日跌出前 20 or 
   板块龙头断板退潮 or 
   个股跌破 20 日线:
    卖出信号 = True
```

**仓位管理**：
- 单只股票：≤ 总仓位 15%
- 单一板块：≤ 总仓位 25%
- 总仓位：40-50%

**持有周期**：1-3 个月

**预期收益**：年化 25-45%

---

### 2. 趋势跟踪模块（30-40% 仓位）

**核心逻辑**：
- 均线多头排列（5/10/20/60 日线）
- 中期趋势向上（20 日线斜率>0）
- 成交量温和放大
- 突破关键阻力位

**技术指标**：

| 指标 | 参数 | 信号 |
|------|------|------|
| **均线** | MA5/10/20/60 | 多头排列 |
| **MACD** | 12,26,9 | 金叉/水上 |
| **RSI** | 14 | 50-70（强势区） |
| **成交量** | 5 日均量 | +20% 以上 |

**买入信号**：
```python
def check_trend_signal(stock):
    # 均线多头排列
    ma_aligned = (MA5 > MA10 > MA20 > MA60)
    
    # 20 日线向上
    ma20_up = MA20_slope > 0
    
    # MACD 金叉
    macd_golden = MACD_DIF > MACD_DEA
    
    # 成交量放大
    volume_up = volume > MA5_volume * 1.2
    
    # RSI 强势区
    rsi_strong = 50 < RSI < 70
    
    return ma_aligned and ma20_up and macd_golden and volume_up and rsi_strong
```

**卖出信号**：
```python
def check_exit_signal(stock):
    # 跌破 20 日线
    break_down = close < MA20
    
    # MACD 死叉
    macd_dead = MACD_DIF < MACD_DEA
    
    # 趋势破坏
    trend_broken = MA5 < MA10 < MA20
    
    return break_down or macd_dead or trend_broken
```

**仓位管理**：
- 单只股票：≤ 总仓位 12%
- 单一行业：≤ 总仓位 20%
- 总仓位：30-40%

**持有周期**：2 周 -2 个月

**预期收益**：年化 15-30%

## 📈 风控规则

### 单只股票风控

**上限**：
- 趋势轮动：≤ 总仓位 15%
- 趋势跟踪：≤ 总仓位 12%

**止损**：
- 趋势轮动：板块退潮或跌破 20 日线止损（-8%）
- 趋势跟踪：跌破 20 日线或 MACD 死叉止损（-8%）

### 模块风控

**趋势轮动**：
- 单一板块 ≤ 总仓位 25%
- 持仓股票数：3-5 只
- 板块退潮时快速减仓

**趋势跟踪**：
- 单一行业 ≤ 总仓位 20%
- 持仓股票数：5-8 只
- 趋势破坏时及时退出

### 总体风控

**最大回撤控制**：
- 单月回撤 > 10%：降至 60% 仓位
- 单月回撤 > 15%：降至 40% 仓位
- 单月回撤 > 20%：降至 20% 仓位

**仓位上限**：
- 正常情况：70-80%
- 高风险时期：40-60%
- 极端行情：20-40%
