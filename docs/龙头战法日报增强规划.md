# 龙头战法日报功能增强规划

## 📋 目标

优化每日报告，提供**明确的龙头板块推荐、龙头股推荐、仓位建议**，指导实际交易操作。

## 🎯 日报核心模块

### 模块 1：情绪周期判断

**输出内容**：
```
【市场情绪】
情绪评分：65/100  [█████████████░░░░░░░] 📈
情绪阶段：发酵期
涨跌停比：85 : 12
连板高度：5 板
操作建议：加仓主线，积极参与
仓位建议：龙头战法仓位使用 80%（总仓位 16%）
```

**数据来源**：
- 涨停家数（pywencai）
- 跌停家数（pywencai）
- 连板高度（AKShare 涨停池）
- 炸板率（计算）

**判断标准**：
```python
情绪评分 = 50 + (涨停数 - 跌停数) * 2 + 连板数 * 5

情绪阶段:
- 80-100: 高潮期 → 仓位 50-60%
- 60-80:  发酵期 → 仓位 70-80%
- 40-60:  分歧期 → 仓位 40-50%
- 0-40:   退潮期 → 仓位 0-20%
```

**仓位计算公式**：
```python
# 根据情绪阶段确定总使用比例
if 情绪 >= 80: 龙头仓位使用 = 60%  # 总仓位 12%
elif 情绪 >= 60: 龙头仓位使用 = 80%  # 总仓位 16%
elif 情绪 >= 40: 龙头仓位使用 = 50%  # 总仓位 10%
else: 龙头仓位使用 = 20%  # 总仓位 4%

# 仓位分配
主线龙头 = 龙头仓位 * 50%
板块龙头 = 龙头仓位 * 30%
补涨龙 = 龙头仓位 * 20%
```

## 🔧 代码实现计划

### 步骤 1：数据获取增强

**新增函数**：
```python
def get_limit_up_pool(date=None):
    """获取涨停池数据（AKShare）"""
    import akshare as ak
    if date is None:
        date = datetime.now().strftime('%Y%m%d')
    zt_df = ak.stock_zt_pool_em(date=date)
    return zt_df

def get_sector_limit_up_stats():
    """统计各板块涨停股数量"""
    zt_df = get_limit_up_pool()
    sector_counts = zt_df['所属板块'].value_counts()
    return sector_counts

def calculate_leader_score(stock_data):
    """计算龙头股综合评分"""
    score = (
        stock_data['板块地位分'] * 35 +
        stock_data['连板高度分'] * 25 +
        stock_data['涨停时间分'] * 20 +
        stock_data['带动效应分'] * 15 +
        stock_data['抗跌性分'] * 5
    )
    return score
```

### 步骤 2：龙头识别逻辑

**新增函数**：
```python
def identify_sector_leaders(sector_name):
    """识别板块内龙头股"""
    # 获取板块内所有涨停股
    zt_df = get_limit_up_pool()
    sector_stocks = zt_df[zt_df['所属板块'] == sector_name]
    
    # 计算每只股票的评分
    leaders = []
    for _, stock in sector_stocks.iterrows():
        score = calculate_leader_score(stock)
        leaders.append({
            '代码': stock['代码'],
            '名称': stock['名称'],
            '连板数': stock['连板数'],
            '涨停时间': stock['涨停时间'],
            '封单金额': stock['封单金额'],
            '评分': score
        })
    
    # 按评分排序
    leaders.sort(key=lambda x: x['评分'], reverse=True)
    return leaders
```

```python
def generate_evening_report(self):
    """生成晚报（增强版）"""
    # 1. 获取市场情绪
    sentiment = self.get_market_sentiment()
    
    # 2. 获取主线板块
    sectors = self.get_main_line_sectors()
    
    # 3. 识别龙头股
    leaders = self.identify_leaders(sectors)
    
    # 4. 计算仓位建议
    position = self.calculate_position(sentiment)
    
    # 5. 生成报告
    report = self.format_report(sentiment, sectors, leaders, position)
    
    # 6. 保存状态
    self.save_leaders(leaders)
    self.save_sectors(sectors)
    
    return report
```

### 步骤 4：报告格式化

**新增格式化函数**：
```python
def format_report(self, sentiment, sectors, leaders, position):
    """格式化报告输出"""
    report = []
    
    # 情绪模块
    report.append("【市场情绪】")
    report.append(f"情绪评分：{sentiment['score']}/100")
    report.append(f"情绪阶段：{sentiment['stage']}")
    report.append(f"仓位建议：龙头仓位使用{position['usage']}%")
    report.append("")
    
    # 主线板块模块
    report.append("【主线板块推荐】")
    for i, sector in enumerate(sectors[:3], 1):
        stars = "★" * int(sector['rating'])
        report.append(f"{i}. {sector['name']} {stars}")
        report.append(f"   连续强势：第{sector['days']}天")
        report.append(f"   推荐等级：{stars}")
    report.append("")
    
    # 龙头股模块
    report.append("【龙头股推荐】")
    for leader in leaders[:5]:
        report.append(f"   {leader['代码']} {leader['名称']}")
        report.append(f"   连板数：{leader['consecutive']}连板")
        report.append(f"   评分：{leader['score']}/100")
        report.append(f"   建议：{leader['advice']}")
    report.append("")
    
    return "\n".join(report)
```
