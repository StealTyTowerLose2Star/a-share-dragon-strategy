# 仓位配置策略说明书

## 📊 总体仓位架构

```
总资金 100% = 
  ├─ 龙头战法策略 20%（1/5）← 本系统负责
  └─ 趋势策略 80%（4/5）← 待开发系统
       ├─ 趋势轮动 40-50%
       └─ 趋势跟踪 30-40%
```

---

## 🐲 龙头战法仓位（20%）

### 定位
- **风险等级**：高风险
- **持有周期**：3-10 天
- **预期收益**：年化 50-100%（高波动）
- **最大回撤**：可能达到 30-50%

### 仓位使用策略

根据情绪周期动态调整**这 20% 仓位**的使用比例：

| 情绪评分 | 情绪阶段 | 使用比例 | 实际仓位 | 操作策略 |
|----------|----------|----------|----------|----------|
| 80-100 | 高潮期 | 50-60% | 10-12% | 持股待涨，不追高 |
| 60-80 | 发酵期 | 70-80% | 14-16% | 加仓主线，积极参与 |
| 40-60 | 分歧期 | 40-50% | 8-10% | 轻仓试错，等待确认 |
| 0-40 | 退潮期 | 0-20% | 0-4% | 空仓防守，等待信号 |

### 参与方式

#### 1. 打板（启动期）
- **时机**：首板/2 板确认
- **仓位**：龙头战法仓位的 20-30%（总仓位 4-6%）
- **目的**：确认龙头身份，试错

#### 2. 低吸（发酵期）⭐ 主要盈利来源
- **时机**：3-5 板，分歧日回踩
- **仓位**：龙头战法仓位的 50-70%（总仓位 10-14%）
- **目的**：主升浪盈利

#### 3. 持股（高潮期）
- **时机**：5-7 板以上
- **仓位**：持有不动
- **目的**：享受主升浪，准备兑现

### 风控规则

**单只股票上限**：
- ≤ 龙头战法仓位的 50%
- 即总仓位的 10%

**止损规则**：
- 板块龙头：跌破 3 日线止损
- 主线龙头：跌破 5 日线止损
- 总龙头：断板后连续 2 日下跌止损

**止盈规则**：
- 达到目标价分批止盈
- 板块进入高潮期后逐步减仓
- 退潮信号出现时清仓

## 📈 趋势策略仓位（80%）

### 定位
- **风险等级**：中风险
- **持有周期**：2 周 -3 个月
- **预期收益**：年化 20-40%（稳健增长）
- **最大回撤**：控制在 15-20% 以内

### 策略模块

#### 1. 趋势轮动（40-50% 仓位）

**核心逻辑**：
- 识别市场主线板块
- 资金流向监测
- 板块轮动节奏把握
- 主线切换时及时调仓

**关键指标**：
- 板块涨幅排名（连续 N 日前 10）
- 板块成交量变化
- 北向资金行业流向
- 板块内涨停股数量

**买入信号**：
```python
if 板块连续 3 日前 10 and 
   成交量放大 and 
   涨停股数量增加:
    加入轮动池
```

**卖出信号**：
```python
if 板块连续 2 日跌出前 20 or 
   龙头股断板退潮:
    减仓或清仓
```

**持有周期**：1-3 个月

**预期收益**：年化 25-45%

---

#### 2. 趋势跟踪（30-40% 仓位）

**核心逻辑**：
- 均线多头排列（5/10/20/60 日线）
- 中期趋势向上（20 日线斜率>0）
- 成交量温和放大
- 突破关键阻力位

**技术指标**：
- MA5/10/20/60 多头排列
- MACD 金叉/水上
- RSI 50-70（强势区）
- 成交量相对 5 日均量 +20% 以上

**买入信号**：
```python
if 均线多头 and 
   MACD 金叉 and 
   成交量放大 and 
   突破关键位:
    买入信号 = True
```

**卖出信号**：
```python
if 跌破 20 日线 or 
   MACD 死叉 or 
   趋势破坏:
    卖出信号 = True
```

**持有周期**：2 周 -2 个月

**预期收益**：年化 15-30%

## 🎯 风险控制

### 独立风控原则

**三个系统独立风控**：
- 龙头战法、趋势轮动、趋势跟踪独立运行
- 单个策略亏损不影响其他策略
- 各自执行止损，不互相干扰

### 总体风控红线

**单只股票上限**：
- 龙头战法：≤ 总仓位 10%
- 趋势轮动：≤ 总仓位 15%
- 趋势跟踪：≤ 总仓位 12%
- 单只股票总计：≤ 总仓位 20%

**单一板块上限**：
- 龙头战法：≤ 总仓位 15%
- 趋势轮动：≤ 总仓位 25%
- 趋势跟踪：≤ 总仓位 20%
- 单一板块总计：≤ 总仓位 35%

**总仓位上限**：
- 任何时候 ≤ 95%
- 保留 5% 现金应对极端情况

### 极端行情应对

**市场崩盘（上证指数单周 -10% 以上）**：
- 龙头战法：清仓（0%）
- 趋势轮动：降至 20%
- 趋势跟踪：降至 10%
- 总仓位：30%

**市场狂热（上证指数单月 +20% 以上）**：
- 龙头战法：降至 10%
- 趋势轮动：降至 30%
- 趋势跟踪：降至 40%
- 总仓位：80%

## 📝 执行建议

### 每日操作

**龙头战法**：
1. 查看情绪评分
2. 调整仓位使用比例
3. 跟踪持仓股表现
4. 执行止损/止盈

**趋势策略**：
1. 每日收盘后更新趋势信号
2. 检查持仓股趋势状态
3. 执行趋势止损/止盈

### 定期复盘

**每周**：
- 统计本周收益
- 检查仓位配置
- 评估策略有效性
- 调整趋势轮动方向

**每月**：
- 再平衡仓位
- 调整策略权重
- 总结得失

**每季**：
- 全面复盘
- 优化策略参数
- 学习市场新变化

---

## ⚠️ 重要提示

1. **龙头战法是高风险策略**，只用 20% 仓位参与
2. **趋势策略是主力**，保证整体收益稳定性
3. **严格执行风控**，不因为短期盈利放松警惕
4. **三个策略独立运行**，互不干扰
5. **持续学习优化**，市场在变，策略也要进化
